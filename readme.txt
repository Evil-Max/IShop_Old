# IShop
Тестовое задание
Задача: реализовать интернет магазин.
Сервис должен обеспечивать:
1. Отображение списка позиций с возможностью фильтрации по выбранным атрибутам;
2. Добавление позиции в корзину клиента;
3. Оформление покупки и доставки товара.

Конечный результат: war-артефакт

Система сборки проекта Maven;
Проект реализован на основе Spring Boot;
База данных Postgres;
Пользовательский интерфейс html, javascript, bootstrap.
